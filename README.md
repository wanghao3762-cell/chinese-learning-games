# Chinese Pinyin 中文拼音打字 - ZType Shooter

> 🎮 **在线试玩：** https://wanghao3762-cell.github.io/chinese-learning-games/ztype-shooter/

## 游戏简介

拼音打字射击是一款边玩边学中文拼音的休闲射击游戏。

天空中不断飞下带有汉字和拼音的敌人飞船，玩家需要**快速输入对应的拼音字母**来击落它们。打得越快、越准，分数越高！

## 怎么玩

1. **开始游戏**：选择 HSK 等级（1-6级，数字越大词汇越难）
2. **锁定目标**：按敌人拼音的**第一个字母**即可选中它
3. **连续输入**：继续输入该拼音的剩余字母，全部输入正确即可击落
4. **不要打错**：打错字母会打断连击，但**不会切换目标**（保持锁定）
5. **躲避敌人**：敌人撞到飞船会游戏结束，尽量先击落飞得最近的

## 游戏特色

- ✅ **HSK 1-6 全套词汇库** - 覆盖从入门到高级的 1300+ 词汇
- ✅ **拼音音节分隔显示** - 如 `ni hao` 比 `nihao` 更易辨认
- ✅ **目标锁定机制** - 选中一个敌人后专心打完，不会乱跳
- ✅ **智能词频控制** - 每个词最多出现 3 次，不会一直重复
- ✅ **连击倍率系统** - 连续正确输入可获得分数加成

## 操作说明

| 按键 | 功能 |
|------|------|
| `A-Z` | 输入拼音字母 |
| `ESC` | 暂停/继续 |
| 底部滑条 | 调整游戏速度 |
